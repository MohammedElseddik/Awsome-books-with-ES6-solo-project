# Awsome-books-with-ES6-solo-project

This is a web site for storing books and authors using ES6 syntax and ES6 Modules.

## Major languages

- HTML
- CSS
- JavaScript (ES6 syntax and Modules)

## Getting Started

To get a local copy up and running follow these simple example steps.

### Setup

To get a local copy up and running follow these simple steps.

To setup the Awesome-books-Project, in the repo page:
click on code (dropdown list) > Download as ZIP;
or open terminal of path you want to install project and run this command <br>
`git clone git@github.com:MohammedElseddik/Awsome-books-with-ES6-solo-project.git`
